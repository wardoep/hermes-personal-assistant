 Hermes Personal Assistant

    Hermes is my personal AI assistant system built to help organize information, support day-to-day workflows, and serve as the central brain for a larger set of specialized tools and agents.

    Overview

    This project is part of my broader effort to learn AI tooling, automation, and self-hosted workflows while building systems that are useful in my own daily life.

    Hermes is designed to act as the main conversational layer, while more focused tools and bots handle specialized tasks in the background.

    Current Focus

    - Building a reliable personal AI assistant workflow
    - Improving infrastructure and stability
    - Connecting supporting tools for research, automation, and monitoring
    - Learning more about AI agents through hands-on development

    Related Areas

    - Linux
    - Python
    - AI tooling
    - Automation
    - VPS/self-hosted systems
    - Research workflows

    Project Goal

    The long-term goal is to have Hermes running as a dependable central assistant that helps coordinate information, personal projects, and specialized agent-based tools.

    Status

    Active project. Still evolving as I improve the core setup, supporting tools, and overall workflow.
