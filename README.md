Hermes Personal Assistant

    Hermes is my personal AI assistant system built to support my day-to-day workflows, organize useful information, and act more like a personalized digital brain.

    Overview

    I’ve been building Hermes as a practical assistant that can learn my preferences, remember important context, and help with specific tasks across different areas of my life. The project has also helped me learn more about APIs, CLI-based workflows, SSH, VPS management, automation, and how different tools can work together in a real system.

    What I’ve Built So Far

    - Connected Hermes to my Obsidian note system for better long-term organization
    - Created a personal “about me” / profile file so Hermes can better understand my goals, preferences, and communication style
    - Added skills and memory to improve task-specific help and personalization
    - Used Hermes as an assistant for productivity workflows, including Google Calendar-related help
    - Started shaping Hermes as the central assistant in a broader personal AI system

    What I’ve Learned Through This Project

    - How APIs are used to connect tools and services
    - How CLI tools can be used to build and manage workflows more efficiently
    - How SSH is used to securely access and manage remote systems
    - How a VPS environment can be used to host and run assistant-related tools
    - How memory, task structure, and integrations make an assistant more useful over time

    Current Goals

    - Keep improving Hermes as a reliable everyday assistant
    - Automate parts of my workflows and daily life
    - Add more specialized subagents for focused tasks
    - Expand into assistant-connected bots and tool integrations, including Discord-based workflows

    Why I’m Building It

    I’m building Hermes both as a useful personal system and as a hands-on way to learn more about AI assistants, automation, memory systems, APIs, CLI workflows, and agent-based tools.

    Status

    Active project. The system is already useful in real ways and is continuing to evolve as I add more structure, automation, and specialized tools.
