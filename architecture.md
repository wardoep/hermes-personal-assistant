    Architecture

    Hermes is intended to serve as the main conversational layer in my personal AI setup.

    Core Idea
    - Hermes acts as the central assistant
    - Supporting tools and agents handle more focused tasks
    - The system is being developed incrementally, with stability and usability prioritized first

    Current Components
    - Main assistant workflow
    - VPS-based environment
    - Supporting tools for research and automation
    - Separate project lanes for specialized bots

    Direction
    The system is being built to become a practical everyday tool while also helping me learn more about applied AI systems.
